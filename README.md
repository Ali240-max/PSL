# PSL Management System

A console-based Java application for managing Pakistan Super League (PSL)
cricket data including teams, players, match schedules, and standings.

## Author

Ali Farooqi - BSEE012315001
Noor Fatima - BSSE012315002

## How to Run

- Requires Java JDK 17 or above
- Compile and run MainMenu.java from the src/ directory

## Project Structure

- src/model → Data classes (Player, Team, Stadium etc.)
- src/service → Business logic (PSLSystem, Search, Admin)
- src/ui → Console menu interface
- src/admin → Admin login and management
